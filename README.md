<h1 align="center">Instagram Analytics Dashboard (Power BI)</h1>

<p align="center">
A data-driven Instagram Analytics Dashboard built using Power BI to visualize engagement, reach, and content performance.
</p>

<hr>

<h2> Project Overview</h2>
<p>
This project showcases an <b>interactive Instagram Analytics Dashboard</b> developed using <b>Power BI</b>. 
It provides deep insights into social media performance by analyzing key metrics such as engagement, likes, reach, and follower growth.
The dashboard is designed to support data-driven decision-making for improving content strategy and audience engagement.
</p>

<hr>

<h2> Objectives</h2>
<ul>
  <li>Analyze engagement across different media types (Image, Carousel, Reel)</li>
  <li>Identify high-performing content categories</li>
  <li>Evaluate traffic sources contributing to reach</li>
  <li>Track follower growth trends over time</li>
  <li>Provide actionable insights for social media optimization</li>
</ul>

<hr>

<h2> Dashboard Features</h2>

<h3> Key Performance Indicators (KPIs)</h3>
<ul>
  <li>Total Accounts</li>
  <li>Total Likes</li>
  <li>Total Engagement</li>
  <li>Average Shares</li>
</ul>

<h3> Interactive Visualizations</h3>
<ul>
  <li>Engagement by Media Type</li>
  <li>Followers Gained by Day of Week</li>
  <li>Reach by Traffic Sources</li>
  <li>Likes by Content Category</li>
  <li>Posts Distribution by Media Type</li>
  <li>Engagement Trend by Post Date</li>
</ul>

<hr>

<h2> Tools & Technologies Used</h2>
<ul>
  <li>Power BI</li>
  <li>Power Query (Data Cleaning & Transformation)</li>
  <li>DAX (Data Analysis Expressions)</li>
  <li>Data Modeling</li>
  <li>Interactive Dashboard Design</li>
</ul>

<hr>

<h2> Key Insights</h2>
<ul>
  <li>Reels and carousel posts generate higher engagement compared to static images</li>
  <li>Hashtags and Explore page contribute significantly to overall reach</li>
  <li>Content categories such as Fashion and Photography attract more likes</li>
  <li>Follower growth varies across different days, indicating optimal posting times</li>
</ul>

<hr>

<h2> Project Structure</h2>
<ul>
  <li><b>Dataset:</b> Instagram performance dataset</li>
  <li><b>Dashboard File:</b> Power BI (.pbix)</li>
  <li><b>Visuals:</b> Interactive charts and KPI cards</li>
</ul>

<hr>

<h2> How to Use</h2>
<ol>
  <li>Download the <b>.pbix</b> file from this repository</li>
  <li>Open using <b>Power BI Desktop</b></li>
  <li>Interact with slicers (Day, Traffic Source)</li>
  <li>Explore visuals for insights</li>
</ol>
